# 这是一个为osu设计的音乐播放桌面程序
之前一直用osu听歌，因为其不能在后台切歌以及创建歌单很麻烦；故先用winform写了一个能解析.osu文件的音乐播放器demo

## 预期实现以下功能
1. 后台切歌
2. 随机播放
3. 顺序播放
4. 创建歌单
5. 显示谱面图像
6. 做一个显示音乐波形的动态图像
7. 自定义快捷键  
~~然而很可惜上述设计目标目前一个都没完成~~  
以下是已经做完的部分
8. 显示谱面图像
9. 快捷键切换上下曲（窗口是活动状态）
10. 播放音乐
11. 显示谱面名称和乐曲作者
12. 解析osu的song文件夹，提取出其中音乐信息
